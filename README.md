# Shyne App 🌟

**Tu mejor versión, cada día.**

App de imagen personal con IA. Incluye:
- Onboarding personalizado
- Chat con Luca, tu estilista personal
- Generador de looks con tu armario
- Rutina de skincare personalizada
- Recomendaciones de estilo

## Instalación

```bash
npm install
npm start
```

## Variables de entorno

Crear un archivo `.env` en la raíz:

```
REACT_APP_ANTHROPIC_KEY=tu_api_key_aqui
```

## Deploy

Conectar el repositorio a [Vercel](https://vercel.com) para deploy automático.
